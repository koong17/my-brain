# Decisions

"왜 그렇게 했나" 기록. 1결정 1파일, 파일명: `YYYY-MM-DD-제목.md`

```markdown
---
id: YYYY-MM-DD-제목
kind: decision
status: active          # 뒤집히면 superseded — 수정 말고 새 결정이 supersedes로 가리킴
updated: YYYY-MM-DD
supersedes:             # (선택) 뒤집은 이전 결정 id
---

# <결정 제목>

- 상황: (무슨 문제였나)
- 선택지: (고려한 대안들)
- 결정: (+ 이유)
- 결과: (나중에 돌아보고 추가)
```

쌓이면 `identity/principles.md`의 근거 자료가 된다.
